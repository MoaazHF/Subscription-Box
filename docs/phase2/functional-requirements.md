# Functional Requirements (Final)

1. User registration, login, logout, and role-based authorization.
2. Subscription lifecycle management: create, pause, resume, plan change, renewal.
3. Billing and payment record generation with tax computation.
4. Box provisioning, item swapping, add-ons, lock-date controls, and allergen checks.
5. Bundle selector: reusable predefined sets applied to a box.
6. Delivery lifecycle management with explicit state transitions.
7. Driver assignment and status updates within allowed transition boundaries.
8. Claim creation for damaged/missing deliveries, including photo evidence.
9. Admin claim resolution and rejection workflow with resolver tracking.
10. Notifications queueing and dispatch through channel abstraction.
11. Admin management panels for users, products, subscriptions, drivers, warehouse staff, zones, bundles.
12. Reports generation as printable views and CSV exports.
